运行代码

说明

本功能用于快速运行一段简短的代码。

示例

运行代码 python
msg = 'Hello, world!'
print(msg)

正在运行，请稍等……

stdout:

Hello, world!

格式

运行代码　编程语言
代码

PS: 注意代码前面需要换行

命令别名

run, 运行, 运行代码

编程语言

assembly, bash, c, clojure, coffeescript, cpp, csharp, erlang, fsharp, go, groovy, haskell, java, javascript, julia, kotlin, lua, perl, php, python, ruby, rust, scala, swift, typescript
