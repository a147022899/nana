Pastebin

说明

本功能用于粘贴代码（或其它任意文本）到 Pastebin（目前使用 CCZU Pastebin）。

示例

粘贴 C

#include <stdio.h>

int main() {
    printf("Hello, world!");
    return 0;
}

粘贴成功，链接 https://paste.cczu.org/p/tIYEj8u3

粘贴 -
我能吞下玻璃而不伤身体

粘贴成功，链接 https://paste.cczu.org/p/6YcSFuSY

格式

粘贴　语法名
要粘贴的内容

PS: 注意要粘贴的内容前面需要换行

命令别名

paste, 粘贴, 粘贴代码

语法名

-, Bash, Batchfile, C, C++, C#, CMake, CSS, Cython, D, Delphi, Diff, Erlang, Fortran, Go, Haskell, HTML, Jade, Java, JavaScript, LLVM, Logtalk, Lua, Makefile, Mako, Matlab, MySQL, NASM, Nginx configuration file, NumPy, objdump, Objective-C, OCaml, Perl, PHP, PostScript, Prolog, Properties, Python, Python 3, Ragel, Ruby, Redcode, reStructuredText, Sass, Scala, Scheme, SCSS, SQL, TeX, VB.net, XML, YAML

其中，「-」表示纯文本。其它支持的语法请参考 https://paste.cczu.org/static/about.html。
